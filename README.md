# Mi Proyecto Streamlit

## Descripción
Streamlit de Datasets de Uber

## Instalación
1. Crear entorno virtual: `python -m venv venv`
2. Activar entorno: `venv\Scripts\activate` (Windows) o `source venv/bin/activate` (Mac/Linux)
3. Instalar dependencias: `pip install streamlit pandas numpy matplotlib seaborn folium reportlab`

## Ejecución
`streamlit run app.py`
